Привет всем! Мой сайт теперь завершён. В процессе я освоил новые техники, которыми хотел бы поделиться, а также показать сайты, которые меня вдохновляли.

## Технологические стеки

- [next-intl](https://next-intl-docs.vercel.app)：Поддержка нескольких языков на сайте.
- [Framer motion](https://www.framer.com/motion/)：Полезная библиотека анимаций, которая упрощает создание сложных эффектов анимации, которые сложно реализовать только с помощью TailwindCSS.
- [Neon](https://neon.tech/)：Мой сайт использует платформу postgresql для отслеживания **просмотров постов**.
- [Prisma](https://www.prisma.io/)：Эта ORM-библиотека создана для лёгкого взаимодействия с базами данных MySQL и предлагает интуитивные API, которые упрощают процесс работы с базами данных.
- [Velite](https://velite.js.org/)：Управляет содержанием моих постов в блоге и проектов, и легко позволяет определить нужную информацию для поста, такую как заголовок, описание, дата публикации и т. д.

По моему опыту, дизайн пользовательского интерфейса для вебсайта может занять много времени. Когда мне не хватает вдохновения, следующие сайты предлагают множество вариантов и экономят время на разработке дизайна:

- [Awwwards](https://www.awwwards.com/)
- [Dark Mode Design](https://www.darkmodedesign.com)
- [Godly](https://godly.website/)
- [UI Verse](https://uiverse.io/)
- [Aceternity UI](https://ui.aceternity.com/components)

Кроме того, вот несколько сайтов с богатым функционалом и современным, красивым дизайном UI:

- [leerob.io](https://leerob.io)
